# Guide photo

Save the guide's photo in this folder as **`guide.jpg`** (a square crop looks
best — it's shown in a circle). It appears automatically in:

- the app header, next to the "Eufaula Bite Score" title
- the "Ask the guide" explanation panel

If the file is missing the header shows nothing and the explanation panel
shows a rod-and-reel placeholder — the app works fine either way.

You can delete this file once guide.jpg is in place.
